
# ListlyFlow

A task management/to-do web app made using the Django MVT pattern. 


## Features

- Add task
- Assign task to project/category
- Give priority level to tasks
- Add a due date
- Sort tasks via:
    Category,
    Priority,
    Due Date
- Find specific task
- Search task
- Edit task
- Delete task
- Complete task
- Receive updates on email
